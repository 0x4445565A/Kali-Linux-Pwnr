# Kali-Linux-Pwnr
HID USB Attack designed for Kali Linux systems.

Demo: https://vid.me/e/D7W9?autoplay=1

What the full attack does
=-=-=-=-=-=-=-=-=-=-=-=-=-=

- Opens up Gnome Terminal
- Runs the Python Shell
- Changes background to a skid using Kali
